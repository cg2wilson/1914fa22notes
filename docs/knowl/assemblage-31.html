Comparison Properties of Integrals.

Let \(f,g\) be continuous functions on \([a,b]\text{,}\) and let \(c\) be a point in \([a,b]\) such that \(a\lt c \lt b\text{.}\) Then,

  1. If \(f(x)\geq 0\) for \(a\leq x\leq b\text{,}\) then \(\ds \int_a^b f(x)\,dx\geq 0\)

  2. If \(f(x)\geq g(x)\) for \(a\leq x\leq b\text{,}\) then \(\ds \int_a^b f(x)\,dx \geq \int_a^b g(x)\,dx\)

  3. If \(m\leq f(x)\leq M\) for \(a\leq x\leq b\text{,}\) then \(\ds m(b-a)\leq \int_a^b f(x)\,dx\leq M(b-a)\)
