Section 3.5 Summary of Curve Sketching

Objectives

  • Sketch curves using calculus and other qualitative characteristics of a function (domain, intercepts, asymptotes, symmetry)

Subsection 3.5.1 Before Class

Subsubsection 3.5.1.1 Slant Asymptotes

Definition 3.5.1. Slant Asymptote.

If \(\ds \lim_{x\to \infty} \left[f(x)-(mx+b)\right] = 0\text{,}\) where \(m\neq 0\text{,}\) then the line \(y=mx+b\) is called a slant asymptote.

For rational functions, slant asymptotes occur when the degree of the numerator is one more than the degree of the denominator.

Example 3.5.2.
Find the slant asymptote(s) for the function \(f(x) = \dfrac{3x^2}{x-1}\text{.}\)
Example 3.5.3.
Show that the hyperbola \(x^2-y^2 =1 \) has asymptotes at \(y = \pm x\text{.}\)

Subsubsection 3.5.1.2 Summary of Graphing

Guidelines for Curve Sketching.

  1. Find the domain of the function.

  2. Identify \(x-\) and \(y-\)intercepts.

  3. Locate symmetry in the graph, i.e. determine if the function is even, odd, periodic, etc.

  4. Locate any asymptotes: vertical, horizontal, or slant.

  5. Determine the intervals where the function is increasing or decreasing.

  6. Find local maxima/minima.

  7. Determine the intervals where the function is concave up/concave down.

With this information, you can confidently sketch the graph.

Example 3.5.4.
Sketch the graph of \(f(x) = x^3 + 3x^2\)

Example 3.5.20.

Investigate the family of curves given by the equation \(f(x) = x^4 + cx^2 + x\text{.}\) Start by determining the transitional values of \(c\) at which the number of inflection points changes.
