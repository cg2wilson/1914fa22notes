Section 1.8 Continuity

Objectives

  • State the definition of continuity at a point and use the definition to determine if a function is continuous at a point

  • Identify removable, jump, and infinite discontinuities and give examples of functions that include at least one of these types of discontinuities

  • Define one-sided continuity and connect one-sided continuity to overall continuity at a point.

  • Justify when a function is continuous on an interval by using theorems, limit laws, and the definition of continuity

  • State and use the Intermediate Value Theorem

Subsection 1.8.1 Before Class

Subsubsection 1.8.1.1 The Definition

Definition 1.8.1. Continuity.

A function is continuous at input \(a\) if \(\ds \lim_{x\to a} f(x) = f(a)\)

If this condition fails, then \(f\) is said to be discontinuous at input \(a\).

\(f\) is continuous on an interval if it is continuous at every number in the interval.

Conditions for Continuity.

Continuity of a function requires three things:

  1. \(a\) is in the domain of \(f\)

  2. \(\ds \lim_{x\to a} f(x)\) exists

  3. \(\displaystyle \ds \lim_{x\to a} f(x) = f(a)\)

Definition 1.8.5. Continuity from the Right and Left.
A function \(f\) is said to be continuous from the right at input \(a\) if \(\ds \lim_{x\to a^+} f(x) = f(a)\text{,}\) and \(f\) is continuous from the left at input \(a\) if \(\ds \lim_{x\to a^-} f(x) = f(a)\text{.}\)

Subsubsection 1.8.3.1 Useful Results

Theorem 1.8.10. Properties of Continuous Functions.
If \(f\) and \(g\) are continuous at \(a\text{,}\) and if \(c\) is a constant, then the following functions are all continuous at \(a\text{:}\)

  1. \(\displaystyle f\pm g\)

  2. \(\displaystyle cf\)

  3. \(\displaystyle fg\)

  4. \(\dfrac{f}{g}\text{,}\) for \(g\neq 0\)

  5. \(\displaystyle f\circ g\)

Theorem 1.8.13. Types of Continuous Functions.
The following functions are continuous on their domain:

  • Polynomials

  • Rational Functions

  • Root Functions

  • Trigonometric Functions

Subsubsection 1.8.3.2 The Intermediate Value Theorem

Theorem 1.8.15. Intermediate Value Theorem.

Suppose that \(f\) is continuous on the closed interval \([a,b]\text{,}\) and let \(N\) be any number between \(f(a)\) and \(f(b)\text{,}\) where \(f(a)\neq f(b)\text{.}\) Then, there exists a number \(c\) in \((a,b)\) such that \(f(c) = N\text{.}\)
